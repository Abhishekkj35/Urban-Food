# UrbanFood – Online Food Ordering Platform

UrbanFood is a full-stack food ordering web application that allows users to browse restaurant menus, place food orders, and manage purchases through a seamless online interface. The system supports role-based dashboards for both users and administrators, enabling efficient menu management and order processing.

# Features

Restaurant Menu Browsing

User Authentication System

Online Food Ordering

Cart and Checkout System

Order Tracking and Management

Admin Dashboard for Menu Management

Responsive User Interface

# Tech Stack
Frontend: React.js, Tailwind CSS, JavaScript

Backend: Node.js, Express.js, REST APIs

Database: MongoDB, Mongoose

Payment Integration: Stripe

Other Tools: Git, GitHub

# Application Workflow

User visits the homepage.

User browses restaurant menu items.

User logs into the platform.

User selects food items and adds them to the cart.

User proceeds to checkout and completes payment.

Admin manages menu listings and tracks incoming orders.
